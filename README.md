# Netflix Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

Project Tasks
- Data cleaning and preprocessing
- Analysis of Movies vs TV Shows
- Country-wise content distribution
- Release year trends
- Genre analysis

Key Insights
- Netflix contains more movies than TV shows.
- The United States produces the highest amount of Netflix content.
- Content production increased significantly after 2015.
- Drama and International Movies are among the most common genres.

Dataset Source
Kaggle – Netflix Movies and TV Shows Dataset
